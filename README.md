<h1>pln_post_processing_jatis</h1>

This is the code associated with the paper : "Analysis of Angular-Differential Post-Processing Algorithms for Exoplanet Direct Detection with a Photonic Lantern Nuller". 
The main directory contains the Jupyter notebook, `pln_post_processioing.ipynb`, that reproduces the figures from the paper. The `requirements.txt` and `environment.yml` give the relevant OS, versioning, and modules used to make these code and materials public.

<h1> Regarding the Simulated Data and the Localization Results </h1>

There are simulated data files, and dataset localization results, that are required to fully run through the `pln_post_processioing.ipynb` notebook, but since most of them exceed the Github file size for allowed commits, they can be found on the Zenodo release of this repository with the DOI: 10.5281/zenodo.20835372 .
